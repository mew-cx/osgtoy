Hi Robert,

Here's a new pseudo-loader, this one creates planet globes from cylindrical
image maps (see sample images below), e.g.:

        osgviewer venmap2.gif.1.globe

Or recreate a nursery rhyme :-) using the moon image available from
http://gw.marketingden.com/planets/images/moonmaps/moonmap1k.jpg

        osgviewer moonmap1k.jpg.10.globe cow.osg.0,0,15.trans

cheers
-- mew
